setPivotToAlign

Boolean

Defined in setPivotToAlign:18

Determines if the text object's pivot property will reflect the text's alignment, e.g. a BitmapText with align of 'right' will have pivot.x set to the text's width, so that the right edge of the text is at the text's position. Setting to false uses PIXI's default alignment.

Default: false
